# [BOJ] 1339 단어 수학

## 문제 전략

1. **그리디 알고리즘**: 각 알파벳에 0~9 숫자를 할당할 때, 가장 큰 합을 만들기 위해 **가중치가 높은 알파벳에 큰 숫자를 부여**합니다.
2. **자릿수 기반 가중치 계산**: 알파벳이 등장하는 자릿수에 따라 가중치를 계산합니다. 예를 들어, "ABC"에서 A는 $10^2$, B는 $10^1$, C는 $10^0$의 가중치를 갖습니다.
3. **정렬 후 할당**: 모든 알파벳의 가중치를 계산한 뒤, 내림차순 정렬하여 가장 큰 가중치부터 9, 8, 7, ... 순서로 숫자를 할당합니다.

## 해결 과정

1. `weight` 배열을 26개의 0으로 초기화하여 각 알파벳(A~Z)의 가중치를 저장할 공간을 만듭니다.
2. `n`개의 단어를 입력받아 `words` 리스트에 저장합니다.
3. 각 단어를 순회하며 알파벳별 가중치를 계산합니다:
   - `ord(word[i]) - 65`로 알파벳 인덱스를 구합니다.
   - 해당 자리의 가중치 `10^(length - i - 1)`를 누적합니다.
4. `weight.sort(reverse=True)`로 가중치를 내림차순 정렬합니다.
5. 가중치가 0이 아닌 알파벳에 9부터 내림차순으로 숫자를 할당하며 결과를 계산합니다.
6. 최종 결과를 출력합니다.

## 복잡도 분석

### 시간 복잡도: $O(N \times L + 26 \log 26)$
- **가중치 계산**: N개의 단어, 각 단어의 평균 길이 L에 대해 O(N \times L)이 소요됩니다.
- **정렬**: 알파벳 26개를 정렬하는 데 O(26 \log 26), 즉 O(1)이 소요됩니다.
- **결과 계산**: 최대 10개의 알파벳에 대해 O(1)이 소요됩니다.
- **결론**: 전체 시간 복잡도는 **O(N \times L)**입니다.
