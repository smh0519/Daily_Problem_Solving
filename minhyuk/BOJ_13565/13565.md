# [BOJ] 13565 - 침투

## 문제 풀이 전략
격자 형태의 그래프에서 상단의 전류(0)가 하단까지 연결되는지 확인하는 경로 탐색 문제입니다. 격자의 크기가 최대 $1,000 \times 1,000$으로 크기 때문에, 효율적인 탐색을 위해 **DFS(깊이 우선 탐색)**를 선택했습니다.

## 해결 과정
1. **탐색 시작점 설정**: 전류는 바깥쪽(제일 윗줄, y=0)에서 시작하므로, 첫 번째 행의 모든 칸을 순회하며 '0'인 지점에서 탐색을 시작합니다.
2. **그래프 탐색 (DFS)**:
   - 상, 하, 좌, 우 네 방향으로 이동하며 전류가 통할 수 있는 칸('0')을 찾습니다.
   - 격자의 범위를 벗어나거나 벽('1')을 만나면 탐색을 중단합니다.
3. **중복 방문 방지 (Visited)**: 이미 방문한 칸은 다시 방문하지 않도록 값을 '2'와 같은 별도의 숫자로 변경하여 무한 루프를 방지합니다.
4. **성공 조건 확인**: 탐색 중 y 좌표가 마지막 행(height - 1)에 도달하면 전류가 안쪽까지 침투한 것이므로 즉시 `True`를 반환하고 전체 탐색을 종료합니다.
5. **결과 출력**: 단 한 번이라도 바닥에 닿았다면 **"YES"**, 모든 시작점에서 실패했다면 **"NO"**를 출력합니다.

### 복잡도 분석
- **시간 복잡도**: $O(N \times M)$ (격자의 모든 칸을 최대 한 번씩 방문)

##  어려웠던 점 및 복기
- **TypeError (NoneType) 발생**: 초기 구현 시 모든 경로에서 값을 반환(return)하지 않아 int와 None을 연산하려는 에러가 발생했습니다. 모든 조건 분기에서 명확한 반환 값을 설정하는 것이 중요하다는 것을 배웠습니다.
- **방문 처리의 중요성**: 방문 처리를 누락하면 이미 지나온 길을 다시 되돌아가며 무한 재귀에 빠지게 됩니다. DFS/BFS 문제에서 방문 배열 혹은 원본 배열 수정을 통한 방문 표시는 필수임을 다시 확인했습니다.
